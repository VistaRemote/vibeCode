# VistaRemote 与 LuminaryWorks 生态

> **品牌**：VistaRemote · **中文名**：视界远程  
> **组织**：[github.com/VistaRemote](https://github.com/VistaRemote) · **MetaRepo**：[VistaRemote/vibeCode](https://github.com/VistaRemote/vibeCode)

**Slogan**：Self-hostable WebRTC remote desktop + AI recording insights.

## 独立价值

VistaRemote 可**单独交付**：跨平台远程桌面控制、会话录制、自托管 AI 摘要，适配工控、IT 运维、客服场景。

## 在 AI 生态中的角色

| 维度 | 说明 |
|------|------|
| **控** | 人工触达现场：设备桌面、边缘网关、工控机 |
| **AI 洞察** | 录制 → 摘要/异常检测 → DataLuminary 报表（可选） |
| **与 VistaCast 分工** | VistaRemote = 远程桌面；[VistaCast](https://github.com/VistaCast/vistacast) = 固定摄像头 AI（并存） |

```text
告警/人工 ──► VistaRemote WebRTC ──► 录制 ──► AI 分析 ──► DataLuminary（可选）
```

## 兄弟产品

| 产品 | 集成场景 |
|------|----------|
| [SyncroBrain](https://github.com/syncrobrain/LuminaryIoTChain) | 设备远程维护 |
| [DataLuminary](https://github.com/dataluminary/DataLuminary-Platform) | 运维报表与审计大屏 |
| [BlockyEdu](https://github.com/blockyedu/VibeEdu) | WebRTC 远程运维实验 |

| [VistaCast](https://github.com/VistaCast/vistacast) | 摄像头 AI 告警 → 可选远程介入 |
